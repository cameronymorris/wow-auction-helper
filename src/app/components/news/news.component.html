Welcome back!

New functionality have been added since your last visit, among them youl'l find the following:

Crafting updates
  • You can now set up a list of your crafting characters in the settings. This allows you to only display recipes you know, instead of every recipe in the database.
  • You can now choose what items you want to override the price of manually. This setting is found in the settings menu. You simply search for the item hit enter and click on the icon to add an item. Thus change it's value and click "Save all changes".

This message will only be displayed upon feature updates and not bug fixes. If you find any bugs or wish to give feedback on the app, please do so on in the reddit thread. As always, any feedback is appreciated!
