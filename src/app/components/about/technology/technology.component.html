The web client

The client is created using Angular v.5. My intention is to keep it at the latest release.

The application uses Angular Material combined with Bootstrap 4 for it's styling. Some custom modifications has been made also.

Server side

The current backend is currently written in PHP. But I intend on moving it to NodeJS and ExpressJS.

For the database that stores all the recipes, items and pets it uses MariaDB. It is probably going to be replaced later on, with DynamoDB.

May I see the code?

All the code for the client is available on GitHub.

The new backend is also available in the same repository, but not the php backend.
