About

WAH - Crafting and auctioning tool is developed with the purpose of making it easier to earn gold in World of Warcraft. It helps the user figuring out the crafting cost of a target item. It also allows you to browse the auction house, get Blood of Sargeras value, find out what items are below vendor sell value and more!

If you got any feedback or any issues with the application, do please report then on the reddit thread. Click here in order to get there, or search for it manually. If you don't report issues, they most likely might not be discovered. If you find any and report them, Il'l fix them as soon as I have time.

Technologies:

The source code is available over at GitHub, you may feel free to fork it and create something based on it if you wish. But please do leave credit, where credit is due. This webapplication uses the following technologies:
  • Angular 4
  • Php
  • MySQL
For the stylesheet, bootstrap is used along with two themed versions that can be found at bootswatch. The dark theme (default) is Solar and the light theme is Paper.
